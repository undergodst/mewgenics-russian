# Mewgenics — Русский перевод (Machine Translation)

Неофициальный русский перевод Mewgenics. Перевод выполнен автоматически через Google Translate и может содержать неточности.

## Установка

**Требования:** Python 3.10+ ([скачать](https://www.python.org/downloads/))

1. Скачайте и распакуйте архив
2. Запустите `install.bat`
3. Скрипт автоматически найдёт игру и установит перевод
4. Запустите игру через Steam

## Удаление

1. Запустите `uninstall.bat`
2. Или проверьте целостность файлов через Steam

## Обновление игры

После обновления игры Steam может перезаписать `resources.gpak`. Просто запустите `install.bat` снова.

## Известные проблемы

- Машинный перевод — некоторые фразы звучат неестественно
- Имена способностей и предметов могут быть переведены буквально
- Шрифт может отличаться от оригинального
